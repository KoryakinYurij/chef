# Technologies

**Backend:**
- **Convex:** A reactive database and serverless platform for building web applications.
- **Node.js:** The runtime environment for the backend and agent.

**Frontend:**
- **Remix:** A full-stack web framework for React.
- **React:** A JavaScript library for building user interfaces.
- **TypeScript:** A statically typed superset of JavaScript.
- **Vite:** A fast build tool and development server.
- **Tailwind CSS:** A utility-first CSS framework.

**AI Integration:**
- **AI SDK:** A library for interacting with various AI models.
- **Providers:** Supports models from OpenAI, Anthropic, Google, and XAI.

**Key Dependencies:**
- **`@remix-run/*`:** Core packages for the Remix framework.
- **`convex`:** The client library for interacting with the Convex backend.
- **`@ai-sdk/*`:** Libraries for integrating with different AI model providers.
- **`react` & `react-dom`:** For building the user interface.
- **`vitest`:** For running unit and integration tests.
- **`pnpm`:** Used as the package manager for the project.

**Development & Tooling:**
- **ESLint & Prettier:** For code linting and formatting.
- **`start-server-and-test`:** For running end-to-end tests that require a development server.
